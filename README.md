# 트리플 과제

트리플에서 진행하는 과제입니다.
트리플 홈페이지의 메인화면 중 한 섹션을 구현해야합니다.
여러 섹션 중에서도 트리플 앱 어워드 수상한 섹션을 구현합시다.

# 설치하기

```bash
npm install
```

# 실행 방법

```bash
npm run start
```

# 프로젝트 기술 스택

```
react v18.2.0 (2022.07.02 기준 creact-react-app 최신 버전으로 진행)
sass(scss 사용): v1.53.0
 - css 선택자의 중복 코드 제거 (ex. child 선택을 위한 parent > child 구조 코드)
 - css 보다 좋은 가독성
```
